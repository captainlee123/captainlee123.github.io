轨迹跟踪开发文档

一、手动部署

文件导入

引入libMapToolsSDK.a 和 include头文件夹

设置定位权限

  • Target --> Info --> NSLocationAlwaysAndWhenInUseUsageDescription
  • Target --> Info --> NSLocationWhenInUseUsageDescription

二、使用方法

引入头文件

#import <HYMapToolsSDK/HYMapToolsSDK.h>

获取定位信息

[[HYLocationManager shareInstance] startLocationCompletion:^(HYLocationObj *locationObj) {
    //返回定位信息
}];

坐标转换

此SDK提供常用坐标系之间的转换：

  • WGS84（标准坐标）
  • 火星坐标
  • 墨卡托（投影坐标）
  • BD-09（百度坐标）
  • GCJ-02（中国国测局坐标）
// 例：WGS84转BD-09
[HYCoordinateTransform wgs84ToBd09:<#(CLLocationCoordinate2D)#>];
